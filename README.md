# my-webpack-project
基于vue 2.x + webpack 4.x的demo，萌新必备
## 首要任务
首先你需要安装Node.js，点击打开Node.js下载页面，安装v8.5之后的版本。安装完Node.js后，会自带npm包管理器。
如果这两个你都有了就直接走下一步☟☟☟
## 安装依赖
> npm install
## 生产环境
> npm run build
## 开发环境
> npm start / npm run dev
## 打开浏览器
地址输入localhost:8088  就可以看到项目运行成功了
